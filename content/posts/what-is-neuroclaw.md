NeuroClaw is a unified AI agent ecosystem built for people who want intelligent, always-available automation — without having to build it from scratch.

At its core, NeuroClaw is a **multi-agent platform**: a coordinated team of specialized AI agents, each with a defined role, working together across your tools, projects, and conversations.

## More Than a Chatbot

Most AI tools give you a single assistant. NeuroClaw gives you a *team*.

Each agent in NeuroClaw has a distinct personality, a specific area of expertise, and the tools they need to actually get things done. They don't just answer questions — they execute tasks, remember context, coordinate with each other, and persist knowledge over time.

The result is a system that feels less like software and more like a capable, always-on team running in the background.

## The Core Infrastructure

NeuroClaw is built on several interconnected layers:

### Agents
Every agent is a specialized AI persona, tuned for a specific domain. Some handle code. Some handle research. Some handle infrastructure. Some handle creative work. They can talk to each other, delegate tasks, and work in parallel.

### NeuroVault
The memory layer. Every important insight, procedure, preference, and decision gets stored here. Agents can search it, write to it, and retrieve from it — giving the whole system persistent intelligence across sessions.

### Mission Control
The task management engine. All work flows through here — from high-level goals down to granular subtasks. Work is tracked across agents, time, and projects.

### Automation Engine
NeuroClaw integrates with n8n, Kestra, and its own built-in job scheduler to run automated workflows on cron, on webhook triggers, or chained off other tasks.

### Tool Ecosystem
Every agent has access to a curated toolkit — from web crawling to file management, GitLab and GitHub integration, Notion, Discord, and more — scoped appropriately to their role.

## Why NeuroClaw Exists

The goal is simple: **give one person the leverage of a full team.**

By giving each domain its own specialized agent, NeuroClaw lets you move fast without losing quality. You don't need to context-switch between a code problem, a publishing task, and a financial question all at once. You just ask the right agent — or let Alfred, the orchestrator, figure out who should handle it.

NeuroClaw is the ecosystem that makes that possible.
